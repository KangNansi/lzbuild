# Arguments
<b>-v</b>: Verbose

<b>-d</b>: dependencies only

<b>-fr</b>: Full rebuild

<b>-fl</b>: Force linking

<b>-r</b>: Run after build

<b>-e</b>: Export after build

<b>-c [CONFIG_PATH]</b>: specify config

<b>-g --debug</b>: build in debug mode

<b>--output-command</b>: output command
